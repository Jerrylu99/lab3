# Guatemala Economic Dashboard (World Bank Data)

## Overview
This project pulls macroeconomic indicators from the World Bank API and builds a 2x3 executive dashboard for Guatemala.

## Data Source
World Bank API via `wbgapi`.

## Indicators Used
- GDP_Const
- Inflation_CPI
- Unemployment_Rate
- Tax_Rev_GDP
- Gov_Exp_GDP
- Exports_GDP
- Imports_GDP
- Gross_Dom_Savings
- Gross_Cap_Formation

## How to Run
1. Open the notebook in Google Colab.
2. Run cells in order (data fetch → cleaning → visualization → dashboard).

## Output
A 2x3 executive dashboard titled **"Guatemala Economic Snapshot"**.
