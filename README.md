# Guatemala Economic Dashboard (World Bank Data)

## Overview
This project pulls macroeconomic indicators from the World Bank API and builds a 2x3 executive dashboard for Guatemala.

## Data Source
World Bank API via `wbgapi`.

## Indicators Used
- GDP_Const
- Inflation_CPI
- Unemployment_Rate
- Tax_Rev_GDP
- Gov_Exp_GDP
- Exports_GDP
- Imports_GDP
- Gross_Dom_Savings
- Gross_Cap_Formation
